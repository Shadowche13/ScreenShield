# ScreenShield
Леко и интуитивно десктоп приложение за защита на зрението по време на гейминг или работа. Използва прецизен хардуерен алгоритъм за фина настройка на гамата, елиминирайки синята светлина без изкуствено разваляне на цветовете. Включва родителски контрол с парола, готови потребителски профили, статистика и пълен двуезичен интерфейс.
